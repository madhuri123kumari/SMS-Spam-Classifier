# SMS-Spam-Classifier:-The SMS Spam Classifier is a machine learning project developed to automatically identify whether a text message is spam or legitimate (ham). The project uses Natural Language Processing (NLP) techniques to preprocess and analyze SMS messages. Text data is cleaned through tokenization, stop-word removal, and vectorization methods such as TF-IDF. Multiple machine learning algorithms were tested, and the best-performing model was selected based on accuracy, precision, and recall.
The classifier helps users filter unwanted messages, reducing the risk of scams and improving communication efficiency. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and prediction functionalities. Python libraries such as Pandas, NumPy, Scikit-learn, and NLTK were used during development. This project demonstrates practical applications of machine learning and NLP in real-world spam detection systems and provides an efficient solution for automated SMS classification.
